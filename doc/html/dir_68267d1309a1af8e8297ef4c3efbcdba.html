file  sections.h [code]
 This file contains functions for getting information concerning the section headers (e.g. sections_by_name_is_writable() informs you wether the writable flag in a section header is set). An other type of function changes the values within the section headers (set functions like sections_by_name_set_name() ) or may even add a new one. The functions which convert e.g. pointers to rva values where placed in this header too, because they need information from the section headers to calculate their return values.
